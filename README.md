Blinkit Sales and Performance Dashboard
Project Overview

The Blinkit Sales and Performance Dashboard is a Business Intelligence project developed using Microsoft Power BI to analyze and visualize sales performance, customer behavior, product trends, and outlet efficiency of Blinkit.

This dashboard helps in understanding key business metrics through interactive visualizations and provides valuable insights for better business decision-making.

Objectives
Analyze overall sales performance
Monitor product category performance
Understand customer purchasing behavior
Track outlet performance and sales trends
Identify high-performing and low-performing products
Improve business decision-making using data visualization
